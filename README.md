## <a href="https://swooo.sh">swooo[sh]</a> - A simple URL shortener

#### Fast, reliable & efficient. Create a shortened URL, with a customizable ID (`https://swooo.sh/{id}`).

Example:

`https://swooo.sh/hv` => `https://herbievine.com`
